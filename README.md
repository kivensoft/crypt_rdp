# windows远程桌面连接配置文件密码生成工具

#### 介绍
生成rdp文件的密码

#### 软件架构


#### 安装教程

1.  创建目录 `mkdir build`
2.  进入目录 `cd build`
3.  生成Makefile `cmake -G "Unix Makefiles" ..`
4.  编译 `make`

#### 使用说明

`Usage: crypt_rdp <password>`
